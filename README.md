# Drawing machine

### Parts

#### General

* MDF cutouts
* 4 x M4-60
* 3 x M4-20
* 15 x M4 nuts
* 4 x M3 10mm
* 4 x M3 Nuts
* Breadboard
* Jumper wires
* 4 x zip ties
* 9V battery
* Ball pen

#### Electronics

* Arduino Pro Micro
* Micro Servo
* ZC-A0591 Stepper Driver
* 28BYJ-48 Stepper
* Photodiode
* 10k resistor
